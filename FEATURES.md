## User Workflows

### Basic Workflow
1. Visit the home page
2. Click "Start Creating" to enter the Studio
3. Drag and drop or select media files
4. Click any item to add metadata (title, tags, notes)
5. Generate a collage from your images
6. Download your creation

### Project Management
1. All work is auto-saved continuously
2. Close the browser and come back anytime
3. Click "New Project" to start fresh
4. Confirm before clearing current work

### Collage Creation
1. Add multiple images to your project
2. Navigate to "Create Something Fun" section
3. Click "Generate Collage"
4. Preview the result
5. Download or regenerate for variations
